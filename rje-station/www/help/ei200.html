EI200 Commands
AGAIN | A,nnnBackspace print file nnn (octal) sectors, or rewind if nnn is 0.
CONTINUE | CResume printing after suspend or interrupt.
DROP | D,snDrop job with sequence number sn.
END | E,LPStop currently printing file and drop it.
END | E,CRTerminate job deck begin read, and release it from mass storage.
GO | GRestart card reading after suspension.
IGNORE | IIgnore interrupt key.
LOGIN | L,idIdentify the terminal to the system. This command must be issued prior to any other action at the terminal. id is the user number for the terminal.
READ | RBegin reading cards, or perform a GO function if reading in progress.
SUSPEND | SHalt a currently printing file and hold its position until further commands are given.
