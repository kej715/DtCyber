RBF Commands
ABORT,devAbort current output on specified device.
CHANGE,JOB=jsnSelect specific job to be the next one to print instead of using the normal queue order.
DISPLAYShow all devices.
DISPLAY,PRShow print queue.
DISPLAY,EXShow executing jobs.
DISPLAY,INShow input queue.
DISPLAY,PUShow punch queue.
DISPLAY,JOB=jsnDisplay specific job status.
GO,devStart specified device.
PURGE,JOB=jsn,ALLPurge jsn from all queues. Use PR, PU, PT, IN, or EX instead of all for a specific queue. If the job is executing it will cancel the job.
REWIND,dev/td>Rewind output on specified device.
SKIP,dev,ENDSkip to end of output on specified device.
SKIP,dev,DFLSkip forward to dayfile portion of output on specified device.
STOP,devTemporarily stop specified device.
